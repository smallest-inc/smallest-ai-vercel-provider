# Smallest AI Provider

The [Smallest AI provider](https://github.com/smallest-inc/smallest-ai-vercel-provider)
for the AI SDK contains support for text-to-speech (Lightning v3.1 and
Lightning v3.1 Pro) and speech-to-text (Pulse and Pulse Pro) models.

For more information about Smallest AI, see the
[Smallest AI Documentation](https://docs.smallest.ai).

## Setup

The Smallest AI provider is available in the `smallestai-vercel-provider` module.

## Provider Instance

You can import the default provider instance `smallestai` from `smallestai-vercel-provider`:

```ts
import { smallestai } from 'smallestai-vercel-provider';
```

If you need a custom configuration, use `createSmallestAI`:

```ts
import { createSmallestAI } from 'smallestai-vercel-provider';

const smallestai = createSmallestAI({
  apiKey: process.env.SMALLEST_API_KEY, // default
});
```

## Speech Models

You can create Smallest AI speech models using the `.speech()` method. The
package also exports `DEFAULT_LIGHTNING_MODEL` so you don't have to hard-code
the model id:

```ts
import { generateSpeech } from 'ai';
import {
  smallestai,
  DEFAULT_LIGHTNING_MODEL,
} from 'smallestai-vercel-provider';

const { audio } = await generateSpeech({
  model: smallestai.speech(DEFAULT_LIGHTNING_MODEL),
  text: 'Hello from Smallest AI!',
  voice: 'sophia',
});

// audio.uint8Array — raw audio bytes (WAV)
// audio.base64 — base64-encoded audio
```

### Speech Model Options

| Model ID | Description |
| --- | --- |
| `lightning_v3.1` | Default. 44.1 kHz, natural expressive speech, sub-100ms latency, 12 languages with auto-detect, voice cloning supported |
| `lightning_v3.1_pro` | Premium broadcast-quality pool (English + Hindi), American / British / Indian accents. No cloning |

### Voices

Smallest AI provides 200+ voices across multiple languages and accents.
Pass any catalog `voice` as a string, and **pair it with its pool** —
standard voices use `lightning_v3.1`, Pro voices require
`lightning_v3.1_pro`.

Standard pool (`lightning_v3.1`):

| Voice | Gender | Accent | Best For |
| --- | --- | --- | --- |
| `sophia` | Female | American | General use (default) |
| `robert` | Male | American | Professional, reports |
| `advika` | Female | Indian | Hindi, code-switching |
| `vivaan` | Male | Indian | Bilingual English/Hindi |
| `camilla` | Female | Spanish | Spanish content |

Pro pool (`lightning_v3.1_pro`): `meher` (Indian), `sophie` (British), `blake` (American), and more.

### Provider Options

You can pass provider-specific options using the `providerOptions` parameter:

```ts
const { audio } = await generateSpeech({
  model: smallestai.speech(DEFAULT_LIGHTNING_MODEL),
  text: 'Hello!',
  voice: 'robert',
  providerOptions: {
    smallestai: {
      sampleRate: 24000,         // 8000 | 16000 | 24000 | 44100
      outputFormat: 'mp3',       // pcm | mp3 | wav | mulaw (→ ulaw) | alaw | ulaw
      pronunciationDicts: ['<dict-id>'],
    },
  },
});
```

## Transcription Models

You can create Smallest AI transcription models using the `.transcription()` method:

| Model ID | Languages | Notes |
| --- | --- | --- |
| `pulse` | 38 (auto-detect) | Batch and WebSocket streaming (64 ms TTFT) |
| `pulse-pro` | English only | Batch/pre-recorded only — leaderboard-ranked accuracy; no streaming |

```ts
import { transcribe } from 'ai';
import { smallestai } from 'smallestai-vercel-provider';
import { readFileSync } from 'fs';

const { text, segments } = await transcribe({
  model: smallestai.transcription('pulse'),
  audio: readFileSync('recording.wav'),
  mediaType: 'audio/wav',
});
```

### Transcription Provider Options

```ts
const result = await transcribe({
  model: smallestai.transcription('pulse'),
  audio: audioBuffer,
  mediaType: 'audio/wav',
  providerOptions: {
    smallestai: {
      language: 'multi',         // 'en' | 'hi' | 'multi' | 30+ ISO codes
      diarize: true,             // speaker identification
      emotionDetection: true,
      genderDetection: true,
      wordTimestamps: true,

      // Privacy
      redactPii: true,           // names, addresses → [FIRSTNAME_1] etc.
      redactPci: true,           // card #s, CVV → [CREDITCARDCVV_1] etc.

      // Formatting
      numerals: 'auto',          // 'true' | 'false' | 'auto'
      punctuate: true,
      capitalize: true,

      // Keyword boosting (max 100; "WORD:INTENSIFIER")
      keywords: ['NVIDIA:5', 'Jensen:4'],

      // Async webhook delivery
      webhookUrl: 'https://example.com/asr-webhook',
      webhookMethod: 'POST',
      webhookExtra: 'job_id:abc123',
    },
  },
});
```

> Note: `ageDetection` has been removed from the server API and emits a deprecation warning if set.
